### What I'm building

**[doable](https://github.com/sammasak/doable)**
AI coding interface — describe what you want to build, watch it happen. Backed by an autonomous Claude VM runtime. Built with Svelte + TypeScript.

**[playground](https://github.com/sammasak/playground)**
Rust chess engine compiled to WebAssembly. Bot plugin system using the [WASM Component Model](https://component-model.bytecodealliance.org/) — write bots in Rust or Python. In-browser code editor via CodeMirror 6. 97 PERFT-verified test cases.

**[nixos-config](https://github.com/sammasak/nixos-config)**
Fully declarative homelab: NixOS hosts, Home Manager, k3s cluster, Flux GitOps, KubeVirt workstation VMs, SOPS-encrypted secrets. Reproducible from scratch.

**[claude-code-skills](https://github.com/sammasak/claude-code-skills)**
Custom skills, agents, and hooks for Claude Code — delivered via Nix Home Manager across all machines.
